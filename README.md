# DED Challenge

## Requirements
1. Docker
2. Net 5 SDK

## RUN
Just run docker compose in the root of the project. 
 
```
 docker-compose up 
```

## Stack Used 
1. CQRS using Mediator (MediatR)
2. Vertical Slice Architecture
3. Net 5.0
4. NServiceBus
5. ASP.Net Core 5 Web Api
6. ASP.Net Core Blazor Server Side
8. MongoDB
9. RabbitMQ
10. Docker
